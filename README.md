## Qui_est_ce

## Installation
* Run `make all` (`make build` + `make run`) from qui_est_ce directory (where **makefile** is located)

## JSON Datasets
3 basic sets are already provided : **Animals**, **Human**, and **Countries**.
JSON Structure :

```
    .
    └── theme
        ├── human
        |   └── objects
        |       ├───.
        |       |   ├── id: name
        |       |   ├── ...
        |       |   └── attributes
        |       |       ├── attribute1
        |       |       └── ...
        |       |
        |       └── ...
        |
        ├── animals
        |   └── ...
        |   
        └── ...
```
